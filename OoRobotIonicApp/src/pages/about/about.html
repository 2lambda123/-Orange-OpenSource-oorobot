À propos d'OoRoBoT

L'OoRoBoT est un robot OpenSource permettant aux enfants d'appréhender la programmation de façon simple et ludique.

Retrouver toutes les informations à propos du robot sur le site Web du projet : https://orange-opensource.github.io/oorobot/

Vous pouvez réaliser des tracés avec une interface graphique équivalente à cette proposée dans l'onget "code" de cette application sur cette page Web. Vous pouvez charger le programme réalisé sur la page en appuyant sur le bouton "Charger sur l'appli", puis dans cette application vous devez scanner le QRCode avec le bouton de l'onglet "Code"

Vous avez des questions ou des remarques, n'hésitez pas à nous contacter sur oorobot@fablab-lannion.org
